Uniform Resource Locator, is a web address that points to a specific file on the internet. The structure of a URL can include several parts, but it will always have at least three: Scheme or protocol
The network communication protocol that connects web browsers and web servers. For example, HTTP or HTTPS. Domain name Also known as a site address, this is what users type into their browser to reach a website. Path Provides extra information to the website address, and is separated by a forward slash (/). Other parts of a URL include: Subdomain Words or phrases that come before the first dot in a URL. For example, "www" is the most common subdomain and indicates that a website is accessible through the internet. Top-level domain The part of the URL that comes after the last dot, such as .com, .org, or .net. This helps identify the type of website and its purpose. Parameters Also known as query strings or URL variables, these are indicated by a question mark (?). URLs can use letters, numbers, and some special characters, but website owners must convert other characters to code.
